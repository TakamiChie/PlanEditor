# PlanEditor
計画書作成用スプレッドシートエディタ

現在作成中。

## 使い方

### 実行
以下のコマンドでとりあえず実行可能です。

```
> npm install
> .\node_modules\electron\dist\electron.exe .
```

Visual Studio Code用のlaunch.jsonを設定しているので、Visual Studio Code上からデバッグ実行することも可能です。

### Windows用のexeファイルを作成する

Windows上では、以下のコマンドで実行ファイルを作成可能です。

```
> node build.js
```
